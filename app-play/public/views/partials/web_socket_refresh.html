Trifecta provides a couple of configuration properties that allows users to control the frequency of topic and consumer group offset updates:
  • trifecta.web.push.interval.consumer defines the interval (in seconds) that changes to consumer offsets will be pushed to web-socket clients
  • trifecta.web.push.interval.topic defines the interval (in seconds) that changes to topics (new messages) will be pushed to web-socket clients
